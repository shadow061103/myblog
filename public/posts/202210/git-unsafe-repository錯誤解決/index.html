git unsafe repository錯誤解決

October 27, 2022

  • git

在下git指令的時候會跳出錯誤 原因是git更新的安全修正 要解決可以用它上面的指令 或是去找gitconfig(路徑:C:/[user][user name].gitconfig) 加上這一段

[safe]
	directory = *
