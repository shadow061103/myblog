git merge 不直接合併的方法

October 27, 2022

  • git

  • git merge member --no-commit --no-ff
    • –no-commit就是不產生一個commit,資料會放到暫存區
    • –no-ff不要fast forward
  • 不commit的話可以git reset --merge或git merge --abort
