docker container操作

October 24, 2022

  • docker

進container改檔案

方法一

1.docker exec -it bash 2.apt-get update 3.apt-get install vim 4.vim <檔名> 5.用insert模式修改，完成後Esc + :wq

方法二

1.docker exec -it bash 看配置跟檔案位置 2.docker cp :<file path+file name> 3.改完檔案內容之後複製回去 docker cp <host file path +file name> :

刪除container

docker container rm [hash]

刪除image

docker image rm [hash]
